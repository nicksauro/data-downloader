## 2. Escopo

- **Em escopo:**
  - Confirmar implementação fiel de **ADR-014 §Layers + §Mock DLL** —
    mock DLL extraído para subpackage opt-in canônico, fake clock
    determinístico ns-exact, layered fixtures.
  - Verificar **6 INV core cobertas por property tests Hypothesis**
    (INV-1, INV-2, INV-3, INV-7, INV-9, INV-11) com >= 100 examples
    each.
  - Confirmar **subpackage `data_downloader.testing/`** exposto cleanly
    para downstream consumers (Epic 4 multi-asset + projetos derivados).
  - Verificar **backwards compat** — `benchmarks/fixtures/mock_dll.py` é
    stub DEPRECATED que re-exporta do canônico (zero breakage).
  - Confirmar **meta-test guard-rail** anti-drift entre
    `INVARIANTS_TESTS.md` e suite Hypothesis (`tests/integration/test_invariants_core.py`).
  - Verificar **fidelidade do mock** ao contrato real (`MockProfitDLL`
    espelha `ProfitDLL.wrapper.ProfitDLL` superfície pública).

- **Fora de escopo:**
  - Auditoria Nelo da fidelidade do mock vs DLL real (cobre
    `tests/unit/test_mock_dll.py::test_mock_surface_matches_real_wrapper`
    + meta-test). Nelo audit síncrono pendente (não-bloqueante para gate).
  - Hypothesis profile `ci` / `dev` formal (deferred Story 2.11/Epic 3
    quando @devops fechar CI gate).
  - F-Q-1 (cobertura `--cov` bloqueada por duckdb) — não resolvido nesta
    story (deferred Story 3.x).

- **Arquivos tocados (validação manual):**
  - `testing/__init__.py` (~52 linhas) — re-exports estáveis
    (`MockProfitDLL`, `FakeClock`, `MockCall`, `TradeRecordSpec`).
  - `testing/mock_dll.py` (~520 linhas) — `MockProfitDLL` extraído com
    Q02-E reconnect quirk + INV-1 violation detection (`callback_violations`
    list).
  - `testing/fake_clock.py` (~270 linhas) — `FakeClock` ns-exact +
    freeze/thaw + `patch_time` context manager.
  - `testing/fixtures.py` (~165 linhas) — fixtures pytest reutilizáveis
    (Layer 1: `mock_dll_session`, `fake_clock`, `tmp_catalog`, `tmp_data_dir`;
    Layer 2: `synthetic_trades_factory`).
  - `tests/property/test_invariants_core.py` — Hypothesis core consolidado
    (6 INVs).
  - `tests/integration/test_invariants_core.py` — meta-test guard-rail
    anti-drift.
  - `tests/conftest.py` — re-exporta de `data_downloader.testing.fixtures`.
  - `benchmarks/fixtures/mock_dll.py` — DEPRECATED stub re-export.
