- **Mudou public_api?** **NÃO.** `src/data_downloader/_updater/` é
  package privado (prefixo `_` ADR-011 convenção) — **zero exports
  via `data_downloader.public_api`**. UI consome via SettingsScreen
  apenas. CLI `data-downloader self-update` deferred V1.1 (D3
  COUNCIL-30). **SemVer impact: NONE** — `__api_version__ = "1.0.0"`
  permanece intacto.

- **Mudou fronteira interna?** **SIM (NEW).** Sub-pacote privado
  `src/data_downloader/_updater/` cria fronteira nova entre
  app (`SettingsScreen`) e auto-updater. Encapsulada — UI consume via
  signal `update_status_changed`. Network I/O isolado em
  `UpdaterStub._fetch_json` (urllib + GitHub API + 5s timeout +
  HTTPS-only). Threading guidance no docstring (R11 — UI deve invocar
  em QThread/QTimer.singleShot para não bloquear MainThread).

- **F-A-3** (UpdaterStub auth-less GitHub API) — Informativo: GitHub
  Releases API auth-less = 60 req/h por IP. Aceitável V1.0 porque
