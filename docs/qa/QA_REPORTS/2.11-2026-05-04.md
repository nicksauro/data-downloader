**Justificativa:**

A Story 2.11 entrega exatamente o que prometeu — **hierarquia de exceptions
ADR-011 em 3 camadas isoladas + adapter pattern + finding H10
(`DownloadHandle.cancel()`) closure cooperativo**. **8/8 ACs PASS literal**.

**Highlights desta gate:**

1. **3 camadas isoladas conforme ADR-011 §Decisão Opção A** — L1
   (`_internal/exceptions.py`, 8 subclasses + marker), L2
   (`_internal/exception_adapter.py`, lookup + decorator + late import),
   L3 (`public_api/exceptions.py`, 2 tipos novos + property
   `humanized_message`).
2. **Property test Hypothesis no-leak** — `tests/property/test_no_internal_leak.py`
   100 examples cobrindo 8 subclasses + 1 future subclass + marker
   presence. Invariante "internals NUNCA vazam em public_api"
   garantida mecanicamente.
3. **H10 closure cooperativo** — `cancel(*, timeout=30.0) -> bool`
   graceful drain entre chunks (preserva INV-12 + R5 + idempotência);
   `result()` raise `OperationCancelled` (Pythonic — força tratar);
   `peek_result()` non-blocking utilitário; `cancelled()` /
   `is_cancelled` non-blocking probe para UI.
4. **Backward-compatible** — `cancel_event=None` default no
   `Orchestrator.run` preserva 12 testes integração pré-existentes;
   chamadas legadas `download(...)` continuam funcionando.
5. **Microcopy Uma authority** — 4 IDs novos + 4 aliases UPPER_SNAKE
   para `humanized_message`. Sign-off COUNCIL-17 §D3.
6. **SemVer impact MINOR aditivo** — Aria classificou (0.3.0 → 0.4.0
   recomendado antes de Epic 3 release). Soft-break em `result()`
   aceito porque cancel real era findings H10 (não funcionava antes).
7. **Felix UNBLOCKED para Epic 3** — UI consome `cancel()` real
   (não placeholder) + mapping `exception type → NL_*` via
   `humanized_message`. Note em COUNCIL-12 §Pendências P1.
8. **76 testes novos PASS, 0 regressão.** Lint ruff + mypy strict
   limpos. ADR-011 amendment 2026-05-04 marca status `accepted (implemented in Story 2.11)`.

**Esta gate desbloqueia:**

- **Epic 2 close (G-Quality-Final)** — uma das condições era
  "ADR-011 accepted + implementada"; **satisfeita**.
- **Felix Epic 3 UI** — cancel API real disponível; mapping
  `exception type → microcopy NL_*` estável.
- **Future SemVer bump** — 0.3.0 → 0.4.0 antes de Epic 3 release.
