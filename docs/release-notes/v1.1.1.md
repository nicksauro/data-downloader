# data-downloader v1.1.1 — hotfix: crash do app GUI ao baixar

> **v1.1.0 + correção do crash da UI ao baixar histórico.** O CLI nunca foi afetado.
> Build `9100017` · 2026-05-12.

## O que mudou

O app GUI (`data_downloader.exe`) crashava ao clicar **"Baixar"** — abortava em `Qt6Core.dll` (`0xc0000409`), precedido de `CreateDataLoader Failed | Access violation` no `Erro.log` da ProfitDLL e `WM_LOGIN_RESULT`/`WM_VALID_LICENSE` perdidos. Duas causas, ambas corrigidas (Story 4.19, tasks #21/#21b):

1. **Duplo init da ProfitDLL no mesmo processo (Q08-E).** "Testar Conexão" inicializava + finalizava a DLL; "Baixar" inicializava de novo — a ProfitDLL Classic **não é re-inicializável** (`Finalize` não zera o estado interno). Fix: `get_dll()` agora mantém uma única instância da DLL por processo; "Testar Conexão" e "Baixar" reusam a mesma; `finalize()` só no encerramento do app. *Bug latente desde v1.0.5* (quando "Testar Conexão" foi adicionado).
2. **Frozen windowed (`console=False`) → stdio inválido.** Em modo windowed, `sys.stdout`/`sys.stderr` eram `None` e os handles OS de stdio ficavam inválidos → a DLL Delphi crashava ao tocá-los. O CLI (`console=True`) nunca foi afetado. Fix: `_ensure_valid_stdio()` na 1ª linha de `main()` reabre os 3 std fds para `os.devnull` antes de qualquer uso da DLL. (Q-DRIFT-39, novo.)
3. **Modo de init alinhado** (#21b): `resolve_dll_init_mode()` é a fonte única do modo (`minimal_handshake`) — "Testar Conexão" e "Baixar" sempre concordam, então o singleton reusa sem descasamento; warning `dll.session.mode_mismatch` se algum caller pedir modo divergente.

## Como atualizar

- **Tem v1.0.x ou v1.1.0 instalado?** Baixe `data-downloader-Setup-v1.1.1.exe` e execute — InnoSetup faz upgrade in-place; `~\.data-downloader\.env` e os dados Parquet são preservados.
- **Instalação nova?** Setup.exe → configure as 3 variáveis em `~\.data-downloader\.env` (`PROFITDLL_KEY` / `PROFITDLL_USER` / `PROFITDLL_PASS`) → rode `data_downloader-cli.exe --healthcheck` (deve dizer `healthcheck OK`) → abra o app.
- Alternativa portable: `data-downloader-v1.1.1-win64.zip` (extraia e rode `data_downloader.exe`).
