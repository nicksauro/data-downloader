# Story 4.12 — Toggle Mostrar/Ocultar segredos validação

## Contexto

`_toggle_secret` (settings_screen.py:592-600) implementa toggle
EchoMode.Password ↔ Normal e troca texto botão. Implementação parece
correta no código, mas **não foi smoke-testado por Pichau** ainda.

Audit preventivo:

- ✅ `_secrets_visible` é `dict[str, bool]` por var (correto — toggle
  individual, não global).
- ✅ Botão muda texto via `BTN_SHOW_SECRET` ↔ `BTN_HIDE_SECRET`.
- ⚠️ Acessibilidade: faltam `setAccessibleName` /
  `setAccessibleDescription` para screen readers.
- ⚠️ Estado visível NÃO persiste entre sessões (intencional? — best
  practice é sempre default OCULTO ao reabrir; validar com Uma).
- ⚠️ Não há shortcut keyboard (alt+M para "Mostrar"?).

## User Story

**As a** usuário com problema de digitação ou screen reader

**I want** toggle "Mostrar segredo" funcional, com label acessível
e default seguro (oculto ao abrir)

**So that** consigo verificar visualmente o que digitei sem expor
credenciais inadvertidamente.

## Acceptance Criteria

1. **AC1 — Smoke Pichau confirma toggle funciona.**
   - Abre Settings → digita PROFITDLL_KEY → clica "Mostrar" → vê
     valor em plain text → clica "Ocultar" → mascarado novamente.

2. **AC2 — Default sempre OCULTO ao abrir Settings.**
   - Mesmo se sessão anterior estava com "Mostrar" ativo, ao
     reabrir Settings (Ctrl+,) ou app, default é EchoMode.Password.

3. **AC3 — Acessibilidade básica.**
   - `value_edit.setAccessibleName(f"{var_name} (segredo)")`.
   - `toggle_btn.setAccessibleName(f"Mostrar {var_name}")`.

4. **AC4 — Auto-hide após 30s.**
   - Se usuário deixou "Mostrar" ativo e está inativo 30s, volta
     automaticamente para `Password` mode (timer per-var).
   - Microcopy hint: "Auto-ocultar em 30s".

5. **AC5 — Não há keyboard shortcut adicional necessário V1.0.3.**
   - Tab + Space funciona via Qt default — suficiente.

## Out of Scope

- Mascaramento parcial (mostrar só últimos 4 chars) — V1.1.
- Logging de "secret revealed" para auditoria — V2.

## Risks

| Risco | Mitigação |
|-------|-----------|
| Auto-hide é intrusivo se usuário está digitando | Reset timer no textEdited; só auto-hide quando idle. |
